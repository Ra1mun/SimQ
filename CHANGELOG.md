## [1.0.0] - 2024-11-04

### Добавлено

#### Backend (SimQ.Core, SimQ.WebApi)

- Базовая архитектура на основе Clean Architecture
- REST API для управления задачами моделирования
- Поддержка MongoDB для хранения данных
- Модуль моделирования систем массового обслуживания (СМО)
- Реализация основных типов агентов:
  - SourceAgent (источник заявок)
  - ServiceAgent (канал обслуживания)
  - QueueAgent (очередь)
  - SinkAgent (приемник)
- Система событийного моделирования через Supervisor
- Сбор и анализ статистики моделирования
- Контроль точности через погрешность генерации (Kolmogorov distance)
- Поддержка различных типов распределений:
  - Экспоненциальное
  - Равномерное
  - Нормальное
- AutoMapper для преобразования моделей
- Swagger/OpenAPI документация
- Health checks для мониторинга
- Логирование через ILogger
- CORS поддержка для frontend

#### Frontend (React)

- Веб-интерфейс на React 18 с TypeScript
- Material-UI компоненты для UI
- Три основные вкладки:
  - Управление задачами (Problems)
  - Просмотр процессов (Tasks)
  - Библиотека агентов (Agents)
- Визуализация графов с D3.js и Dagre-D3
- API клиенты для взаимодействия с backend
- Адаптивный дизайн

- Factory Pattern для создания агентов
- Настройка конфигурации через appsettings.json
- Поддержка различных окружений (Development, Production)

## Semantic Versioning

Проект следует семантическому версионированию (MAJOR.MINOR.PATCH):

- **MAJOR** — несовместимые изменения API
- **MINOR** — новая функциональность с обратной совместимостью
- **PATCH** — исправления багов с обратной совместимостью
